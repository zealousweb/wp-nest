# WPNest AI Instructions

Apply **only** the rules needed for the current task.

## Rule Selection

- **Architecture** → Always.
- **PHP** → PHP files only.
- **ACF Blocks** → Creating or modifying ACF blocks.
- **SCSS** → SCSS changes only.
- **JavaScript** → JavaScript changes only.
- **Accessibility** → HTML, forms, or UI changes.
- **SEO** → Templates, metadata, or content structure.
- **Toolchain** → Validate before completing the task.

## General

- Do not apply unrelated rules.
- Do not make unrelated code changes.
- Reuse existing project patterns.
- Prefer the smallest applicable rule set.
